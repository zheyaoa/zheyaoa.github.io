slot插槽的理解与使用

发表于 2019-03-04 | 分类于 JavaScript ， Vue.js

最近在工作中用到了slot插槽,而以前在学习Vue的过程对插槽的使用比较少。所以趁着周末学习了一下插槽的相关内容，在这里做总结和记录。

​ 在这里关于Vue.js中Vue的基本使用与组件的基础内容这里将不再讲解。如果想对Vue有基本的了解请移步官方文档

什么是插槽

​ 关于什么是插槽在官方文档中的解释并不是非常的清晰。个人的理解是Vue中的插槽是一个占位符。当其父组件被使用时，可将插槽的内容替换。被替换的内容会出现在组件的对应位置。该内容可以使一段字符串，HTML模板或者是一个组件。
